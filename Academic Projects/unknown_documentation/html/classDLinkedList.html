Detailed Description

template<typename eltType>
class DLinkedList< eltType >

Author: Christian Carreras (with thanks to Dr. Spiegel for the original single linked list) Course: CSC 237 Filename: DLinkedList.h Purpose: The double linked list class uses nodes with a forward and backward pointer to make a list easy to iterate through. Also included is the DLinkedList Iterator. The iterator is named DListItr and is used to access data within the list.
