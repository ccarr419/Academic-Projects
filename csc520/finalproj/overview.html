CSC520 Advanced Object Oriented Design: Final Project

Library System Implementation

Author: Christian Carreras
Due Date: 05/07/2018
Course Number: CSC 520
Course Name: Advanced Object Oriented Design
Professor Name: Dr. Dylan Schwesinger
Assignment: Final
Major: CSC Masters Software Development
University: Kutztown University of Pennsylvania
Purpose: To fully implement the code-based equivalent of a system specified in an analysis and subsystem diagram. Data is persistent
and can be added to and deleted at any time of system operation. The system allows members to log in and perform multiple tasks.
Members can view their account details, borrowed books, reservations, change their password and make a reservation. Both members
and non-members can browse and search through the catalog. A client is provided for integration testing of all subsystems.
